title: Описание сущностей и их атрибутов
---

Projects (Проекты) предназначена для хранения информации об импортированной API-спецификации, вокруг которой строится тестирование. Каждый проект соответствует одному файлу OpenAPI. Проект является корневой сущностью и связывается отношением «один ко многим» с сущностью TestRun. Описание атрибутов приведено в таблице 4.

Таблица 4 - Описание атрибутов сущности "Проекты"

| Наименование атрибута | Тип данных | Описание                            |
|-----------------------|------------|-------------------------------------|
| ID                    | integer    | первичный ключ                      |
| Name                  | text       | название проекта                    |
| OpenAPIFilePath       | text       | путь к исходному файлу спецификации |

TestRuns (Запуски тестов) описывает один полный цикл выполнения тестов (нагрузочного и безопасности) над проектом, обеспечивает версионирование и позволяет сравнивать результаты различных прогонов. Запуск тестирования является центральной сущностью и владеет коллекциями Vulnerabilities и SecurityHeadersResults, связанными отношением «один ко многим» через внешние ключи. Описание атрибутов приведено в таблице 5.

Таблица 5 - Описание атрибутов сущности "Запуски тестов"

| Наименование атрибута | Тип данных | Описание                                                                                       |
|-----------------------|------------|------------------------------------------------------------------------------------------------|
| ID                    | integer    | первичный ключ                                                                                 |
| IDProject             | integer    | внешний ключ к сущности Проекты                                                                |
| StartedAt             | text       | время начала запуска теста                                                                     |
| FinishedAt            | text       | время конец запуска теста                                                                      |
| Verdict               | text       | общий вердикт (PASSED, WARNING, FAILED)                                                        |
| LoadTestResultJson    | text       | JSON-представление объекта LoadTestMetric (TotalRequests, AvgResponseTimeMs, ErrorCount и др.) |
| Summary               | text       | краткая текстовая сводка                                                                       |

Сущность Vulnerability (Уязвимость) хранит детальную информацию о каждой уязвимости, обнаруженной модулем тестирования безопасности, позволяет воспроизвести проблему и отслеживать её появление в разных запусках.

Таблица 6 - Описание атрибутов сущности "Уязвимости"

| Наименование атрибута | Тип данных | Описание                                           |
|-----------------------|------------|----------------------------------------------------|
| ID                    | integer    | первичный ключ                                     |
| IDTestRun             | integer    | внешний ключ к сущности Запуски тестов             |
| Type                  | text       | тип уязвимости                                     |
| Severity              | text       | критичность (Critical, High, Medium, Low, Info)    |
| Endpoint              | text       | эндпоинт, на котором обнаружена уязвимость         |
| Payload               | text       | вредоносная нагрузка (если применимо)              |
| Evidence              | text       | фрагмент ответа, подтверждающий наличие уязвимости |
| DetectedAt            | text       | временная метка обнаружения                        |

 SecurityHeadersResults (Результаты проверки заголовков безопасности) содержит результаты пассивной проверки наличия и корректности HTTP-заголовков, связанных с безопасностью. Эта сущность позволяет не только фиксировать факт наличия заголовков, но и отслеживать динамику их настройки от запуска к запуску, что важно в процессах непрерывного улучшения безопасности API. Описание атрибутов приведено в таблице 7.

Таблица 7 - Описание атрибутов сущности "Результаты проверки заголовков безопасности"

| Наименование атрибута | Тип данных | Описание                               |
|-----------------------|------------|----------------------------------------|
| ID                    | integer    | первичный ключ                         |
| IDTestRun             | integer    | внешний ключ к сущности Запуски тестов |
| HeaderName            | text       | название проверяемого заголовка        |
| IsPresent             | bool       | наличие заголовка                      |

| ExpectedValue         | text       | ожидаемое значение                     |
