title: Выбор фреймворка для реализации пользовательского интерфейса
---

Для построения интерфейса применяется Avalonia UI 12.0.2 – кроссплатформенный XAML-фреймворк, вдохновленный WPF \[3\]. Ключевые преимущества:

-  декларативное описание интерфейса на XAML с возможностью горячей перезагрузки;

-  встроенная поддержка стилей и тем;

-  эффективный рендеринг, не зависящий от системных библиотек UI конкретной ОС;

-  возможность запуска на Windows, Linux и macOS из одной кодовой базы.

В качестве архитектурного шаблона для UI используется MVVM (Model-View-ViewModel), реализованный с помощью библиотеки CommunityToolkit.Mvvm 8.2.2 Она генерирует исходный код свойств и команд на основе атрибутов, что сокращает объем шаблонного кода и снижает вероятность ошибок.
