title: Выбор технологии для локального хранения результатов
---

В качестве встроенного локального хранилища результатов выбрана SQLite – легковесная реляционная база данных, не требующая установки сервера и хранящая все данные в одном файле. Для взаимодействия с SQLite используется Entity Framework Core с провайдером Microsoft.EntityFrameworkCore.Sqlite. EF Core предоставляет:

-  объектно-реляционное отображение, позволяя работать с данными на языке C#;

-  систему миграций для версионирования схемы базы данных;

-  кроссплатформенность и высокую производительность.
