title: Выбор языка программирования для реализации программного обеспечения
---

В качестве языка разработки для ProofAPI принят C# в версии 3.10.14, поддерживаемой целевой платформой .NET 10. C# изначально проектировался как язык для создания сложных программных систем с чёткой структурой \[2\]. Поддержка интерфейсов, свойств, событий и современных средств инкапсуляции позволяет естественно реализовать многослойную архитектуру ProofAPI, где каждый модуль взаимодействует через интерфейсы, а зависимости внедряются через DI-контейнер.

Компилятор C# и среда выполнения .NET официально поддерживают Windows, Linux и macOS без изменений исходного кода. Это соответствует требованию к ProofAPI – предоставлять единый инструмент для специалистов по безопасности, работающих на разных операционных системах.

Таким образом, C# на платформе .NET 10 является оптимальным выбором для создания высокопроизводительного, надёжного и удобного в сопровождении инструмента тестирования REST API.
