В качестве среды разработки выбран Visual Studio Code – легковесный, кроссплатформенный редактор с широкой поддержкой расширений для C#, отладки, работы с Git. Использование VS Code позволяет вести разработку на любой ОС и легко интегрироваться с инструментами командной строки .NET SDK.

Целевая платформа – .NET 10 с использованием современных возможностей C#. .NET обеспечивает высокую производительность, зрелую экосистему библиотек и полноценную кроссплатформенность (Windows, Linux, macOS) \[4\]. Выбор последней версии позволяет применять новейшие оптимизации компилятора и синтаксические возможности языка.
