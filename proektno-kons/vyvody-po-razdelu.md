В ходе преддипломной практики была выполнена проектно-конструкторская проработка и программная реализация инструмента для нефункционального тестирования безопасности REST API – ProofAPI.

В процессе работы были уточнены исходные данные ВКР, использованы ранее подготовленные аналитические материалы, выбраны средства разработки и спроектирована многоуровневая архитектура системы на основе слабой связанности и внедрения зависимостей. Также были разработаны структура данных, пользовательский интерфейс на базе паттерна MVVM, программные интерфейсы сервисов и основные алгоритмы работы системы: оркестрация тестов, нагрузочное тестирование с эмуляцией виртуальных пользователей, активный фаззинг безопасности, пассивный анализ заголовков и поиск утечек конфиденциальных данных.

Таким образом, цель преддипломной практики достигнута: подготовлен проектно-конструкторский раздел выпускной квалификационной работы, а также реализован ProofAPI, инструмент для нефункционального тестирования безопасности REST API.
