---

Разрабатываемое программное обеспечение – **ProofAPI** – представляет собой кроссплатформенный инструмент для автоматизированного нефункционального тестирования безопасности REST API. ProofAPI предназначен для специалистов по безопасности, разработчиков и тестировщиков, которым необходимо выявлять уязвимости и оценивать производительность API на ранних стадиях жизненного цикла программного обеспечения.

Основные возможности инструмента включают:

-  импорт API‑спецификаций в формате OpenAPI 3.x (JSON/YAML);

-  нагрузочное тестирование с эмуляцией виртуальных пользователей (RPS, латентность, процент ошибок);

-  активный поиск уязвимостей методами фаззинга (SQL‑инъекции, XSS, Path Traversal, XXE и др.);

-  пассивную проверку заголовков безопасности (HSTS, CSP, X‑Frame‑Options и пр.);

-  автоматическое обнаружение утечек конфиденциальных данных в ответах API (PII, платёжная информация);

-  локальное хранение истории тестов с возможностью версионирования и сравнения;

-  генерацию детализированного HTML‑отчёта с графиками, списком уязвимостей и cURL‑командами для воспроизведения каждого дефекта.

Инструмент должен работать на операционных системах Windows, Linux и macOS и не требовать развёртывания серверной части.
