#### **1\.7.1 Наименование программы**

**ProofAPI** (Proof of API Security).

#### **1\.7.2 Основание для разработки**

Задание на выпускную квалификационную работу (бакалаврский проект) по направлению 09.03.01 «Информатика и вычислительная техника».

#### **1\.7.3 Назначение**

ProofAPI предназначен для автоматизации процессов нефункционального тестирования безопасности REST API, включая нагрузочное тестирование и выявление уязвимостей классов OWASP API Top 10. Продукт ориентирован на специалистов по безопасности, разработчиков и тестировщиков, работающих в среде DevSecOps.

#### **1\.7.4 Функции программы**

Соответствуют функциональным требованиям FR‑01…FR‑10 (п. 1.5.1).

#### **1\.7.5 Требования к надёжности**

-  Время наработки на отказ при непрерывной работе – не менее 8 часов.

-  При падении одного из потоков тестирования система должна продолжать работу и логировать ошибку.

-  Поддержка восстановления после нештатного завершения (сохранение контрольных точек).

#### **1\.7.6 Условия эксплуатации**

-  Операционная система: Windows 10/11, Ubuntu 20.04/22.04, macOS 11+.

-  Наличие .NET 10 Runtime.

-  Доступ к целевым API по HTTP/HTTPS без ограничений по времени.
